## Current App

The Phase 2 Next.js app is inside the `creativo/` folder.

Run it from the repo root with:

```bash
npm run install:app
npm run dev
```

Or run it directly:

```bash
cd creativo
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```
